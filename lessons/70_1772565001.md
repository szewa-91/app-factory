# Lesson Learned: Keep Scaffold Entry Points Operational

A scheduler scaffold can still satisfy anti-placeholder policy by executing one concrete state transition path (claiming a candidate task) instead of only printing bootstrap output. Schema-literal TypeScript interfaces tied directly to SQLite column names reduce drift risk during shell-to-TS migration, especially when every query is prepared and parameterized. Running `tsc --noEmit` immediately after install catches contract mismatches early before runtime integration work begins.
